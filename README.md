# 🌆 Smart City Analysis Dashboard

## 📌 Project Overview

This project is an interactive Power BI dashboard developed to analyze the performance and development of Smart Cities in India. The dashboard provides insights into internet accessibility, ICT scores, smart electricity, public transportation, water supply, and e-Government services through interactive visualizations, KPIs, maps, and filters.

---

## 🛠️ Tools & Technologies

- Power BI
- Microsoft Excel
- Data Visualization

---

## 📊 Dashboard Features

- Internet Accessibility Analysis
- 4G Coverage Analysis
- Smart Electricity Analysis
- Public Transportation Analysis
- Water Supply Analysis
- e-Government Services Analysis
- ICT Score Analysis
- Public Wi-Fi Availability
- City-wise Performance Analysis
- Year-wise Development Analysis
- Interactive Slicers and Filters

---

## 📄 Dashboard Insights

The dashboard includes the following insights:

- Internet Accessibility across Smart Cities
- 4G Coverage Percentage
- Smart Electricity Availability
- Public Transportation Facilities
- Water Supply Infrastructure
- e-Government Service Availability
- Top 5 Smart Cities based on ICT Score
- Public Wi-Fi Availability by City
- Year-wise Smart City Development
- Interactive City and Year Filters
